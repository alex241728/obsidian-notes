# Time constructible
## Definition
$T(n)$ is time constructible if there is a machine computing the mapping $1^n \to T(n)$ in time $O(T(n))$.
# Time hierarchy theorem
## Definition
For any time-constructibe $T(n) \geq n$, $TIME[T^c] \not\subset TIME[T]$ where $c > 1$ is a large enough universal constant.
