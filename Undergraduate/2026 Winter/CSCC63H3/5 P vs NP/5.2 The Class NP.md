# Efficiently Verifiable Problems
$L \in \{0,1\}^*$ is verifiable in time T if
$\exists \text{ TM } V \text{ such that }$
- $x \in L \Rightarrow \exists w \in \{0,1\}^{\leq T(|x|)} \ V(x,w)=1$
- $x \not\in L \Rightarrow \forall w \in \{0,1\}^{\leq T(|x|)} \ V(x,w)=0$
- $V(x,w) \text{ runs in time } O(T(|x|))$
## Examples
### 1
Deciding if an integer can be factored is verifiable in polynomial time.
$L = \{n \colon n \text{ is a composite number}\}$

$L = \left\{ G \colon G \text{ is an n-vertex graph with a clique of size } \frac{n}{2} \right\}$
